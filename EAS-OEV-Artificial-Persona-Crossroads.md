# EAS-OEV 与人工人格研究的交叉：因果视界的计算验证

> **状态：** 理论综合 v0.1（2026-07-03）
> **核心主张：** EAS-OEV 框架的因果视界定理可以在人工人格系统上直接检验，无需人类被试
> **认识论定位：** 基于两个独立研究领域的结构同构性，提出新的验证路径

---

## 0. 为什么需要交叉？

### 两个领域的独立发展

**EAS-OEV 框架**：
- 核心定理：因果视界定理（9.10）——蝴蝶效应使深层因果不可辨识
- 推论：自由意志 = 浅层因果的可辨识性；事后叙事 = 深层因果的合理化建构
- 困难：在人类系统上直接验证需要神经科学实验，实践上非常困难

**人工人格研究**（Artificial Personality / AI Persona）：
- 核心问题：如何给AI系统建模、注入、维持一致的"人格"？
- 关键成果：SCOPE框架（八维结构）、大五人格在LLM中的科学建模、决策特征空间的人格分组
- 发现：人口学特征仅解释约1.5%的行为差异，但AI模型会过度放大

### 交叉的必要性

两个领域**独立发现了相同的现象**：
- EAS-OEV：事后叙事不是深层因果的真实原因，而是浅层描述
- 人工人格：人口学特征（类似"事后叙事"）解释力极弱，但被过度使用

这不是巧合——两个领域都在触碰**因果深度的范畴错误**：
- 试图用浅层描述（人格标签、事后解释）来理解深层因果（真实行为机制）
- 这种"范畴错误"是结构性的，由因果视界保证

## 1. 核心连接点

### 连接1：SCOPE框架 vs O/E/V 三层结构

**SCOPE框架**（arxiv 2601.07110）：
- **条件输入维度**（用于建构人格）：人口学、社会行为、人格特质、身份叙事
- **评估维度**（用于检验人格）：价值观、行为模式、职业身份、创造力

**EAS-OEV框架**：
- **O层（可观测）**：人格画像、意识报告
- **E层（涌现）**：因果视界、混沌特性
- **V层（验证）**：事后可检验性

**结构同构**：
```
SCOPE                    EAS-OEV
─────────────────────────────────────
条件输入维度        ⟺    O层（可观测层）
  （人格的建构）          （人格画像的建构）

评估维度            ⟺    V层（验证层）
  （人格的检验）          （人格画像的检验）

（隐含）动力学结构   ⟺    E层（涌现层）
                          （因果视界、混沌特性）
```

**关键洞见**：SCOPE框架的"条件输入 vs 评估"分离，恰好对应EAS-OEV的"O层 vs V层"。两个框架都在说：**人格的建构和人格的检验是不同的事**。

---

### 连接2：人口学的1.5% vs 事后叙事的结构性虚构

**SCOPE框架的关键发现**：
> "人口学特征仅解释约1.5%的行为差异"

**EAS-OEV的推论**：
> "事后叙事不是深层因果的真实原因，而是浅层描述"

**独立验证**：
- 两个领域**独立发现**：浅层描述（人口学/事后解释）与深层因果（真实行为/真实原因）之间有**结构性偏差**
- 这种偏差不是"错误"，而是**因果视界的必然结果**

**形式化连接**：

设人口学标签为 D，真实行为为 B，完整因果链为 C。

人口学解释力弱意味着：
```
I(B; D) << I(B; C)
```

由因果视界定理（定理9.10），当人口学标签到行为的因果深度 k_deep > k_max 时：
```
I(B; D) ≈ 0    （人口学信息在因果视界之外）
```

而完整因果链的深度 k_C < k_max，因此：
```
I(B; C) > 0    （真实因果在视界内）
```

**推论**：人口学解释力弱（1.5%）的根本原因是**因果深度太深**——人口学标签到真实行为的因果链长度超过了因果视界 k_max，因此人口学信息无法捕捉到真实因果。

这解释了为什么"人口学预测行为"几乎无效，但"人格画像（事后叙事）"却有用——人格画像的因果深度更浅（k < k_max），在视界内可辨识。

### 连接3：人格一致性维持 vs 因果视界的时域稳定性

**人工人格研究的核心问题**：
> 如何在长期交互中保持AI系统的人格一致性？

**EAS-OEV的对应**：
> 人格画像 = 事后叙事的时间平均 ⟨N(t)⟩_t

**连接**：
- 人格一致性 = 人格画像在时间上的稳定性
- 人格漂移 = 因果视界随动力学变化（λ_max 变化）
- "价值对齐漂移" = 因果视界的时域演化

**形式化**：
```
人格一致性度量 = 1 - H(N(t) | N(t-1))
              = 人格画像的熵率
```

当熵率低时，人格稳定；当熵率高时，人格漂移。

---

### 连接4：决策特征空间 vs 因果深度谱

**人工人格研究**（MDPI 2025）：
- 从决策序列中提取人格特征
- 用正交特征分组人格
- 建立"人格-行为"的因果关系

**EAS-OEV**：
- 因果深度谱 δ*(k)
- 浅层因果（k < k_max）可辨识
- 深层因果（k > k_max）不可辨识

**连接**：
- 决策特征空间 = 浅层因果的可辨识部分
- 人格分组 = 在因果视界内的行为模式分类
- "人格-行为因果关系" = 浅层因果的因果效应

**推论**：人工人格研究实际上在做**浅层因果的分析**——它成功了，因为浅层因果在视界内，可辨识。

## 2. 新的理论综合：因果视界人格理论

基于以上连接，提出**因果视界人格理论**（Causal Horizon Personality Theory, CHPT）：

### 核心主张

**主张1：人格 = 因果视界内的行为模式**
```
人格 P = {行为模式 b : k(b) < k_max}
```

**主张2：人格画像 = 浅层因果的自我描述**
```
人格画像 N = argmax_{N'} P(N' | 可访问信息)
```
- 不是深层因果的真实原因
- 但是因果视界内的最优描述

**主张3：人格一致性 = 因果视界的时域稳定性**
```
人格一致性 = f(λ_max 的时域变化)
```
- λ_max 稳定 → 人格稳定
- λ_max 漂移 → 人格漂移

**主张4：人格漂移 = 因果视界的结构性变化**
```
人格漂移 ⟺ |Δλ_max| / λ_max > τ_drift
```
其中阈值 τ_drift 定义为因果视界的相对变化率，经验估计 τ_drift ≈ 0.1（10%）。

**解读**：当动力学不稳定性 λ_max 的变化超过10%时，因果视界 k_max 发生显著变化，导致人格画像的稳定性下降。

### 与现有理论的关系

| 理论 | 核心主张 | CHPT 的增量 |
|------|---------|------------|
| 大五人格 | 人格 = 五个维度的组合 | 人格 = 因果视界内的行为模式 |
| SCOPE框架 | 人格 = 八维结构 | 八维结构对应 O/E/V 三层 |
| 社会认知理论 | 人格 = 认知-情感系统 | 认知-情感系统 = 因果视界的动力学 |

## 3. 验证路径：在AI系统上检验因果视界

### 实验1：AI人格的因果视界测量

**目标**：测量AI系统的因果视界 k_max

**方法**：
1. 设计不同因果深度的决策任务 T_k（k = 1, 2, ..., K）
   - **定义"相同任务"**：相同的状态描述 + 相同的选项集 + 相同的约束条件
   - **定义"因果深度 k"**：任务需要追踪的因果链长度（如：预测 k 步后的结果）
2. 在时间点 t₁ 和 t₂ 分别让AI系统执行相同任务 T_k
3. 计算**决策一致性**：
   ```
   Consistency(k) = P(π(t₁) = π(t₂) | T_k)
   ```
   即在相同任务下，两个时刻选择相同选项的概率
4. 找到一致性急剧下降的 k 值 → k_max 的估计

**预测**：
- 存在临界 k_max，当 k < k_max 时一致性高，当 k > k_max 时一致性低
- k_max 与AI系统的"复杂度"正相关（更复杂的系统有更大的因果视界）

### 实验2：人格画像 vs 内部计算

**目标**：检验"人格画像是浅层描述，不是深层原因"

**适用范围**：本实验需要访问AI系统的内部状态，适用于**白盒系统**（如开源模型、可解释模型）。对于黑盒系统（如GPT-4），使用替代方案。

**方法（白盒系统）**：
1. 用SCOPE框架建模AI人格
2. 让AI解释自己的决策（生成"人格画像"）
3. 同时记录AI的内部计算过程（注意力权重、激活模式等）
4. 比较"人格画像"与"内部计算"的相关性

**方法（黑盒系统替代方案）**：
1. 用SCOPE框架建模AI人格
2. 设计**行为探针任务**：探测AI内部决策机制的标准化任务
3. 从外部行为推断内部结构（类似心理学的"认知探针"）
4. 比较"人格画像"与"行为探针结果"的相关性

**预测**：
- 人格画像与内部计算/行为探针的相关性 < 1（结构性偏差）
- 相关性随决策因果深度增加而下降

### 实验3：因果视界与人格一致性

**目标**：检验"人格一致性 = 因果视界的时域稳定性"

**方法**：
1. 在长期交互中跟踪AI系统
2. 定期测量人格画像（用SCOPE框架）
3. 定期测量因果视界 k_max（用实验1的方法）
4. 分析人格一致性与 k_max 时域变化的关系

**预测**：
- 人格一致性高 ⟺ k_max 稳定
- 人格漂移 ⟺ k_max 变化
